# StaySphere

StaySphere is a full-stack property listing web application inspired by platforms like Airbnb.

I built this project to practice and apply real-world backend concepts like authentication, authorization, database relationships, image uploads, and location-based features â€” not just CRUD operations.

The goal was to create something close to a production-ready app while keeping the UI simple and functional.

---

## What you can do with StaySphere

- Sign up and log in securely
- Create property listings with images
- Add detailed property descriptions
- View listings on an interactive map
- Filter listings by category and search
- Leave reviews and ratings on listings
- Edit or delete only your own listings and reviews
- Experience proper validation and error handling

---

## Tech Stack Used

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- Joi (for validation)
- Passport.js (authentication)

**Frontend**
- EJS templates
- Bootstrap
- JavaScript

**Other Tools & Services**
- Cloudinary for image uploads
- Mapbox for maps and geocoding
- Express-session with MongoDB session store
- Flash messages for user feedback

---

## Project Structure

The project follows an MVC-style structure:
- Routes are separated
- Controllers handle business logic
- Models manage database schemas
- Middleware is used for validation and authorization

This makes the codebase easier to maintain and scale.
